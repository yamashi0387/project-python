# project-python
数あてゲームであるヌメロンと、サイコロで行うポーカーのような遊びであるヨットの動きをpythonで書いてみました。

# ヌメロンについて
ランダムに3桁の自然数を生成し、あなたはその生成された数を推測します。

答えと予測した数字について、位置も含めて当たっていればHIT、位置は違うが含まれていればBITEと呼びます。

例えば、123が答えで132が予測した数字であれば1HIT2BITEと返されます。3HITとなるまで予測を繰り返す遊びがヌメロンです。

ここでのヌメロンは、121や222のような同じ数字を含むものも許します。なのでBITEの計算が数字の重複を許さないものと比べて複雑になります。

numeron.pyは2025年春頃に、その複雑性に真正面から向き合ったものになります。

BITE計算の視点を変えて、2026年1月に明快なコードに書き直したものがnumeron2.pyになります。
